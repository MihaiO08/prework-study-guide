Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element (self closing) holds an url via the (src) attribute to display an image on the page.
  • The (a) / anchor element is used to provide the functionality of a clickable word linked to an external/internal url via the href function when wrapped around certain bits of text.
  • The (meta) tag is used to provide certain characteristics such as charset and device content width, mainly related to optimisation be it SEO or device displayability
  • The header holds other elements such as title which is not displayed on the webpage itself but will appear in search engine results and in the browser tab of the page
  • The footer elements holds additional information, at the bottom of the page.
  • The main element is a container for the content of the page.
  • The statement !DOCTYPE html must be added at the top of the page to state that we are working on an HTML file.
  • The (html) tag holds the entirety of the html code for a specific page, the language attribute should be added for optimisation purposes.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border attribute surrounds an element in a border and numerical values can be used to specify its effect.
  • A box-shadow indicates the use of the shadow surounding an element and its position on the Y and X axis.
  • Text-align is used to align the targeted text in a specified way.
  • Color and background-color define the color of text or background of the targeted element.
  • Height and Width specify the dimensions of a targeted element on the page.
  • Font-size is used to specify the size of text.
